# Climate Analysis Project

## 📦 Reproducing the Environment

SDear reviewers you can follow these steps to set up and run the project on your machine.

### 1. Clone the repository

```bash
git clone https://github.com/Richo-Worku/climatechange-week0
cd climatechange-week0
```

### 2. Create a virtual environment

```bash
uv venv .venv
```

### 3. Activate the environment

**Windows (PowerShell):**

```bash
.venv\Scripts\Activate.ps1
```

**Mac/Linux:**

```bash
source .venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run Jupyter Notebook

```bash
jupyter notebook
```

### 6. Open the notebook

Open the `.ipynb` files (e.g., `ethiopia_eda.ipynb`, `compare_countries.ipynb`) and run the cells and you can see my works.
